# ZRG Sales DMS - Babe-Micci

Hệ thống quản lý bán hàng (DMS) hiện đại cho dòng sản phẩm tã bỉm Babe-Micci.

## Tính năng chính
- Quản lý đơn hàng (Orders)
- Quản lý kho hàng (Inventory)
- Quản lý nhân viên thị trường (Sales Force)
- Hệ thống Check-in AI (AI Check-in)
- Bản đồ định vị nhân viên thời gian thực
- Dashboard báo cáo KPI động

## Công nghệ sử dụng
- **Frontend**: React + TypeScript + Vite
- **Styling**: Tailwind CSS + Framer Motion
- **Map**: Leaflet / React-Leaflet
- **Backend**: Google Apps Script (GAS) / Google Sheets
- **Icons**: Lucide React
